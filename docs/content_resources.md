# KAKEHASHI コンテンツ制作用素材ソース一覧

PR TIMESのように、企業の公式プレスリリースや公的機関のライブラリから、メディア掲載用として画像やテキストを合法的に利用（引用・転載）できるソースのリストです。

## A. プレスリリース配信サービス（PR TIMES競合）
企業が「広めてほしい」と費用を払って配信しているため、メディアとしての紹介は歓迎されます。トレンドや新商品の紹介に最適です。

| サービス名 | 特徴 | URL |
| :--- | :--- | :--- |
| **@Press (アットプレス)** | 国内大手。記事の質が高く、ガジェットや新商品系に強い。画像も高画質。 | [https://www.atpress.ne.jp/](https://www.atpress.ne.jp/) |
| **ValuePress! (バリュープレス)** | 中小企業やスタートアップが多い。ニッチでユニークな商材が見つかりやすい。 | [https://www.value-press.com/](https://www.value-press.com/) |
| **共同通信PRワイヤー** | 共同通信グループ。大手企業や海外企業の日本向けリリースが多く、信頼性が高い。 | [https://kyodonewsprwire.jp/](https://kyodonewsprwire.jp/) |
| **Dream News (ドリームニュース)** | コストが安いため、個人事業主や地方企業の隠れた名品リリースが多い。 | [https://www.dreamnews.jp/](https://www.dreamnews.jp/) |
| **Fashion Press** | ※メディアですが、ブランドの公式ルック（高画質）が豊富。一次ソース探しの起点として優秀。 | [https://www.fashion-press.net/](https://www.fashion-press.net/) |

## B. 国・自治体の公式ライブラリ（文化・観光紹介）
日本の魅力を海外に伝える「KAKEHASHI」のコンセプトに最も合致する、高品質かつ権利関係がクリアなソースです。

| サービス名 | 特徴 | URL |
| :--- | :--- | :--- |
| **JNTO デジタルフォトライブラリー** | **【最重要】** 日本政府観光局の公式サイト。プロ撮影の絶景、伝統工芸、料理などの超高画質画像が観光促進目的なら無料。 | [https://www.japan.travel/photo/](https://www.japan.travel/photo/) |
| **農林水産省「afford」** | 日本の「食」特化。寿司、和牛、農産物などのシズル感ある高画質画像が揃う。 | [https://www.maff.go.jp/j/pr/afford/](https://www.maff.go.jp/j/pr/afford/) |
| **お祭りフォトライブラリー** | （ダイドーグループなど）日本の「祭り」に特化。インバウンド人気が高いコンテンツ。 | [https://www.dydo-gh.co.jp/matsuri/photo/](https://www.dydo-gh.co.jp/matsuri/photo/) |
| **各都道府県の観光公式サイト** | 例：「Go Tokyo」「Kyoto City Official Travel Guide」等の「Media / Press」コーナー。 | (各自治体サイト参照) |
| **JAPAN VIDEO TOPICS** | 外務省提供。日本紹介の動画素材。埋め込み利用などが可能。 | [https://web-japan.org/jvt/](https://web-japan.org/jvt/) |

## 💡 戦略的な使い分け

### 1. トレンド・新商品記事
*   **ソース**: PR TIMES, @Press, ValuePress
*   **方法**: 「新発売」「限定」「初上陸」などのキーワードで検索し、リリース素材を使って紹介記事を作成。
*   **メリット**: 最新情報なのでSNSでの拡散（バズ）が狙いやすい。

### 2. 文化・観光ガイド記事（資産型記事）
*   **ソース**: JNTO, 農林水産省, 自治体ライブラリ
*   **方法**: 「日本の城10選」「和牛の部位解説」など、普遍的なテーマの記事を作成。
*   **メリット**: 画像のクオリティが圧倒的に高く、記事の信頼性が増す。検索流入（SEO）で長く読まれる。

---
**※注意点**: 
利用の際は、各サイトの「利用規約（著作権・使用許諾）」を必ず確認してください。「メディアとしての掲載」は基本的にOKですが、「画像の加工禁止」や「クレジット表記必須」などの条件がある場合があります。
